#### 取得 Windows 主機的 IQN

1. 在 Windows 主機上啟動 Microsoft iSCSI 啟動器。

2. 在 [iSCSI 啟動器屬性] 視窗的 [設定] 索引標籤上，選取並複製 [啟動器名稱] 欄位的字串。
 
    ![iSCSI 啟動器屬性](./media/storsimple-get-iqn/HCS_iSCSIInitiatorPropertiesFigureIQN-include.png)

3. 儲存這個字串。
