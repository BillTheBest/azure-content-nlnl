# David Chappell 所撰寫的 DocumentDB 白皮書 

**摘要：**了解 DocumentDB 是 Microsoft Azure 上受管理的 NoSQL 資料庫。在下列情況下，DocumentDB 是理想的資料庫技術選擇：

* 應用程式會從無結構描述的資料模型獲得好處。
* 需要因應許多儲存體和輸送量的需求而調整資料庫。
* 您想要使用熟悉的查詢語言和開發平台。

本白皮書另涵蓋以下基本概念：

* DocumentDB 資料模型。
* 應用程式使用資料的方式。
* 在維持一致性的前提下，應用程式擁有哪些平衡效能的選項。
