# 如何使用 Azure 資源管理員進行自動化

Azure 資源管理員會簡化在 Azure 基礎結構服務上執行的複雜應用程式的部署和管理。許多應用程式需要結合計算、網路和儲存體資源，例如虛擬網路、儲存體帳戶以及具有網路介面的虛擬機器。Azure 資源管理員可讓您使用或建構 JSON 範本，然後以單一應用程式的方式部署這些所有資源，並集中管理。
