# 從不明來源登入
<p>此報告指出已成功登入您的目錄，同時被指派 Microsoft 視為匿名 Proxy IP 位址之用戶端 IP 位址的使用者 (Tor IP)。這些 Proxy 通常由想要隱藏其電腦 IP 位址的使用者使用，而且可能用於惡意意圖。</p><p> 這份報告的結果會顯示使用者從該位址和 Proxy 的 IP 位址成功登入您的目錄的次數。</p>
