#支援的瀏覽器和裝置
Microsoft Azure 預覽入口網站可在所有最新的桌上型電腦和平板裝置與瀏覽器上執行。

### 支援的裝置
預覽入口網站可在最新的 PC、Mac 和平板電腦上順利執行。目前尚不支援行動電話。

### 支援的瀏覽器
可支援下列最新版的瀏覽器：

- Internet Explorer (10 和更新版本)
- Safari (7 和更新版本)
- Chrome (最新版本)
- Firefox (最新版本)

*不支援 Safari 6 和之前的版本。如果您使用的不是 OS X，您可以使用 Chrome、Firefox，或升級至 OS X Mavericks 以取得 Safari 7。*
