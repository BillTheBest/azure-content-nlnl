# 大規模擷取遙測和應用程式事件
 
現代應用程式可在執行時建立大量的資訊。無論是行動應用程式中的使用者動作或是網站上的點選流，此遙測資料對於長期分析和即時健康情況監視或決策支援很有價值。服務匯流排提供擷取與處理此資訊所需的工具。
